# Nombre del proyecto 
# === Preprocesamiento de Datos del Titanic ===

# == OBJETIVO ===
```
Aplicar el flujo de trabajo de Git y GitHub para la gestión de un proyecto de Ciencia de Datos.
Implementar etapas de preprocesamiento de datos con Pandas sobre el dataset del Titanic.
Limpiar el dataset manejando nulos, codificando variables y normalizando valores numéricos.
Generar un conjunto de datos final, limpio y listo para ser usado.
```
